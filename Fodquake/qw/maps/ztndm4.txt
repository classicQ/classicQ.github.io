* Play Information *

Single Player           : No
Cooperative             : No
Deathmatch              : Yes, 7 respawn spots
Player load		: 2 - 6
Difficulty Settings     : No
New Sounds              : No
New Graphics            : Yes, see copyright / permissions

* Misc information *

To make water transparent, type at the console:

r_wateralpha 0.4

This makes sense only if you're using GLQuake or GL QuakeWorld.


* Construction *

Base                    : New level from scratch
Editor(s) used          : Worldcraft 1.5b, ReMipDLX
Build Time              : ~2 week
